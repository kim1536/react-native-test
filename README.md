 ## 실행 순서
```
Step 1: git clone
$ git clone https://gitlab.com/gcnyy/test/react-native-test.git

Step 2: npm install
$ npm i 

Step 3: pod install
$ pod install

Step 4: Start Metro
 $ npm run start

Step 5: Start your application
 $ npm run android

 or

 $ npm run ios

**android의 경우**

androidstudio의 Emulator 실행 후 아래 명령어 실행
android Emulator 실행 방법은 위 Notion주소에서 참조바람
```
